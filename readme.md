# Fountain

> Easy Font Management

Install fountain globally:

```
npm i -g fountain
```

Create a `font.json` file:

Advanced Options

```
"fonts": {
   "config": {
     "charset": "full",
     "path": "fonts"
   },
   "open-sans": "*",
   "lato": "all",
   "roboto": ["thin", "regular", "bold"]
}
```

Simplest Options

```
"fonts": ["open-sans", "lato", "roboto"]
```
