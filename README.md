# Fontr

> Font Dependency Manager

Install Fontr

```sh
npm install -g fontr
```

## How fontr works?

Create a `fontr.json` file where contains a font list:

```json

{
    "fonts": [
        "open-sans",
        "helvetica-neue"
    ],
    "path": "fonts"
}

```

Then run:

```sh
fontr install
```
